## What this does

FiveM Strafe Trainer is a web-based practice utility built around strafe timing drills for FiveM movement patterns. It supports WASD and DWAS input sequences, while showing live BPM data and session statistics so you can keep an eye on pace and consistency during training.

The layout is tuned for quick repetition and immediate feedback. You get a keyboard visualizer, streak and error counters, plus best-run tracking to make it easier to compare sessions over time. Progress is kept in local storage, which means your recent training history stays available in the same browser.

## Included Features

- WASD pattern training for movement rhythm practice
- DWAS pattern training for alternate input sequences
- BPM counter for live tempo feedback
- Best BPM tracking for session comparison
- Streak counter to measure consistent input runs
- Best streak tracking for personal progress checks
- Error counter for missed or incorrect inputs
- Fullscreen mode for a more focused training view
- Keyboard visualizer for real-time key feedback
- Local storage save system for retaining progress in-browser

## Compatibility Notes

FiveM Strafe Trainer is made for browser usage and is centered on FiveM strafe practice workflows. It uses standard desktop keyboard input and relies on browser local storage to save progress.

Known limits:
- Saved data stays in the current browser profile
- Clearing site data removes stored records
- Some browser settings may affect fullscreen behavior
- Keyboard feedback depends on physical key input from the device in use

## FAQ

### How do I begin?
Open the project in a supported browser, pick a training mode, and follow the key prompts shown on screen.

### Is it possible to alternate training types?
Yes. The trainer provides both WASD and DWAS practice modes.

### Will it remember my results?
Yes, it keeps stats in local storage, so your recent data remains in the same browser unless that storage is cleared.

### Why did my scores disappear?
Check whether browser cache, site data, or local storage was removed. Any of those can wipe saved training records.

### Can I change how the interface looks?
Use the available display controls, including fullscreen and visualizer options, to adjust the training view.

### Does it depend on a specific game build?
No. It is aimed at FiveM strafe practice and browser-based execution rather than a particular in-game install.
